创建一个控制器 adonis make:controller ** (选第二个选项)
 get 请求无参 Route.get('/', "SiteController.index") 控制器.方法

 控制器
 index ({request}){
        return 'posts ' + request.input('page')
 }

 路由参数 Route.get("posts/:id", "PostController.show");

 show ({params}){
        return 'posts ' + params.id
 }
